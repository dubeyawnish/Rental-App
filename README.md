# House Rental Application

### THIS APPLICATION HELPS ANYONE FIND HOUSES OR OFFICE SPACE FOR RENT


There are two roles: Owner and Tenant.  Owner can register and start adding all his/her properties that need to be rented.
This is a web application that allows tenants to browse through properties added by multiple owners and show interest in one or more properties. The owners are notified of the tenant's interest for a particular property and then talk to different interested tenants to finalize one of them for renting that property.

Once the tenant has rented a property, he/she completes onboarding and KYC. The owner can see details such as which property is rented to whom since when, with other rent-related information. Similarly, the tenant can view his/her dashboard where he/she can find all previous data along with current property rent information like the next due date, last rent payment date, etc.

Based on the role, the owner and tenant will have limited access to the platform; the owner cannot see and perform tenant operations, and vice versa.


## Technologies Used
> - ReactJS for building user interface
> - Redux for global state management
> - NodeJS for server-side scripting
> - Express for creating API routes
> - MongoDB for storing data
> - Mongoose for interacting with MongoDB
> - Axios for making HTTP requests
> - JSON Web Tokens (JWT) for user authentication
